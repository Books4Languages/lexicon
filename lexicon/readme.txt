== Instalación ==

This section describes how to install the plugin and get it working.

1. Upload the entire folder `lexicon` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to "lexicon Settings" to change the default settings (optional)
